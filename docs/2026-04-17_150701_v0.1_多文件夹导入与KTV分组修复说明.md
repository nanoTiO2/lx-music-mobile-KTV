# 多文件夹导入与 KTV 分组修复说明

时间：2026-04-17 15:07:01（UTC+8）

## 本次目标

1. 本地音乐支持连续导入多个根目录，而不是只保留一个目录。
2. 已导入的根目录支持删除。
3. 导入后列表只展示主体音频，`htdemucs_*` 分轨只在 KTV 场景中显示并切换。

## 架构调整

1. 设置层新增 `list.importMusicDirs: string[]`，保存多个已导入根目录。
2. 保留 `list.importMusicDir` 作为兼容字段和目录选择器默认起点。
3. 扫描层为每个分组补充 `rootDirPath`、`rootName`，页面可区分来自哪个根目录。
4. KTV 分组键改为基于完整规范化文件名，而不再依赖“前缀 + 时间戳”。
