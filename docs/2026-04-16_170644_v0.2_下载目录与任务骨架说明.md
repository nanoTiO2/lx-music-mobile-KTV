# v0.2 下载目录与任务骨架说明

## 本次新增

- 扩展设置项：
  - `download.saveDir`
  - `download.useCustomDir`
- 新增下载任务持久化存储：
  - `@download_task_list`
- 新增下载核心模块：
  - 默认目录解析
  - 自定义目录切换
  - 任务创建、更新、删除、清理
- 下载页从占位页升级为任务页骨架：
  - 展示当前保存目录
  - 选择目录 / 恢复默认目录
  - 展示已持久化的下载任务
  - 清理已完成任务
- 下载页已重新接入主导航，横屏与竖屏首页均可进入。

## 当前默认规则

- 默认下载目录：`/Download/lxmusic`
- 若用户未配置自定义目录，则回退到默认目录
- 创建任务时会按现有 `download.fileName` 模板生成文件名
- 当前任务状态文案固定为“等待下载接线”，表示目录与任务骨架已完成，但真实下载执行尚未接入
