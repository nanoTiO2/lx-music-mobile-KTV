# 2026-04-20 13:55 LRC 长按菜单 / 进度拖动 / 倍速变调接入说明

## 本轮目标

在 LRC 歌词舞台页面中：

1. 长按弹出圆形快捷菜单。
2. 提供上一首、播放/暂停、下一首、KTV 音轨切换。
3. 在同一菜单中加入可拖动进度栏。
4. 接入倍速与变调控制，且与主播放页共用同一套播放器参数。

## 实现方式

- 文件：`src/screens/LyricStage/index.tsx`
- 处理：
  - 长按页面任意位置后，按触点附近弹出圆形快捷菜单。
  - 圆盘保留：
    - `KTV`
    - `上一首`
    - `播放/暂停`
    - `下一首`
  - 圆盘下方新增紧凑控制面板：
    - 进度拖动条
    - 倍速滑杆
    - 变调滑杆

## 复用链路

- 进度拖动：
  - 直接走 `global.app_event.setProgress(...)`
  - 与主播放页和歌词同步逻辑保持一致
- 倍速：
  - 直接调用 `setPlaybackRate(...)`
  - 同步调用歌词播放速率更新逻辑
- 变调：
  - 直接调用 `setPitch(...)`
  - 设置项写回 `player.pitchSemitones`
- KTV：
  - 继续复用现有 `getKtvOptions / switchKtvVariant`
