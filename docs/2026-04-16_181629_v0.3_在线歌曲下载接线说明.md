# v0.3 在线歌曲下载接线说明

## 本次完成

- 在线歌曲长按菜单新增“下载”入口
- 菜单动作已接到真实下载流程：
  - 创建下载任务
  - 获取在线播放 URL
  - 推断下载扩展名
  - 落地到下载目录
  - 更新下载进度、速度、完成/失败状态
- 下载完成后会尝试同名保存 `.lrc` 歌词

## 代码落点

- 下载核心执行：
  - `src/core/download/index.ts`
- 在线歌曲菜单与动作：
  - `src/components/OnlineList/ListMenu.tsx`
  - `src/components/OnlineList/listAction.ts`
  - `src/components/OnlineList/index.tsx`

## 当前行为

- 入口位置：在线列表长按菜单
- 目录来源：
  - 优先使用 `download.saveDir`
  - 否则回退 `Download/lxmusic`
- 任务列表：
  - 下载页可看到新增任务
  - 状态会经过 `Waiting to start -> Resolving url -> Downloading -> Completed / Error`
