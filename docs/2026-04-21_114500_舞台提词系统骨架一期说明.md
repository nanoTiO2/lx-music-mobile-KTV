# 舞台提词系统骨架一期说明

时间：2026-04-21 11:45 +08:00

## 本轮目标

在不推翻当前歌词舞台与播放器二开的前提下，先把“舞台提词系统”后续所有实现都依赖的基础层补齐：

- 提词共享协议
- 文档解析模型
- 顺序点按与 JK 接力状态机
- Android 展示端原型页面
- SQLite 协作日志升级

## 本轮新增

### 1. 提词共享层

新增目录：

- `src/shared/prompt/types.ts`
- `src/shared/prompt/document.ts`
- `src/shared/prompt/stateMachine.ts`
- `src/shared/prompt/index.ts`

当前已经沉淀：

- `CueDocument`
- `PromptCue`
- `PromptStyle`
- `PromptSnapshot`
- `PromptState`
- `PromptCommand`

### 2. 文档解析入口

当前支持把以下文本转为统一 `CueDocument`：

- `.lrc`
- `.srt`
- `.ass`
- `.txt`
- demo / 粘贴类文本

当前策略：

- `.lrc` 保留时间标签并转为 cue
- `.srt` 读取字幕块时间范围
- `.ass` 读取 `Dialogue:` 行
- `.txt` 与粘贴文本按行切句

### 3. 状态机

当前已实现：

- `sequential_tap`
- `jk_relay`
- `UNDO`
- `CLEAR_SCREEN`
- `SET_MODE`
- `SET_STYLE`
- `LOAD_DOCUMENT`

`jk_relay` 规则已经按产品定义固化：

- 空闲时 `J/K` 任一按下即上屏当前待播句并成为 owner
- 非 owner 的另一键按下会立即切到下一句并接管 owner
- 原 owner 松开无效果
- 当前 owner 松开且无人接力时清屏

### 4. 展示端页面原型

新增：

- `src/screens/PromptStage/index.tsx`
- `src/screens/PlayDetail/components/PromptStageBtn.tsx`

当前页面能力：

- 黑底展示端预览
- 当前句 / 下一句 / 后一句列表
- 单行 / 双行 / 三行 / 跑马灯切换
- 镜像切换
- 对齐切换
- 字号缩放
- 顺序点按 / JK 接力演示按钮
- 设备与链路状态占位信息

当前定位是“展示端骨架页”，不是控制端成品。

## 日志数据库升级
