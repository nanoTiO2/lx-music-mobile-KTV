# 2026-04-20 LRC 编码、KTV 增益与歌词舞台修复说明

## 本轮目标

- 修复 LRC 标签回写后文件乱码
- 修复 KTV 音轨“伴奏增益”不生效
- 修复歌词舞台跑马灯不可用
- 扩充“切换颜色”预设，并增强投影可读性
- 收紧歌词舞台行距与控件占位，减少横屏重叠

## 架构拆分

- `src/core/musicProfile.ts`
  - 负责读取、写回 LRC 中的分析标签
  - 本轮改为“按原编码读、按原编码写”，保留 BOM 与换行风格
- `src/plugins/player/utils.ts`
  - 负责 TrackPlayer 与 native mixer 的音量、变调、KTV 分轨控制
  - 本轮补齐 mixer 激活状态下的 KTV 增益重下发
- `src/screens/LyricStage/index.tsx`
  - 负责歌词舞台 UI、跑马灯、投影模式与排版
  - 本轮修正动画开关条件、压缩控制面板高度、清理字体 padding
- `src/shared/lyricStagePresets.ts`
  - 负责舞台主题与字体预设
  - 本轮新增多套颜色主题

## 关键修复

### 1. LRC 写回乱码

- 不再直接用默认文本编码覆盖 `.lrc`
- 增加编码识别与保真写回：
  - `UTF-8`
  - `UTF-8 BOM`
  - `UTF-16 LE/BE`
  - `GB18030`
- 回写后继续保留原文件的换行风格

### 2. KTV 伴奏增益

- mixer 启动本地播放时，改为带上当前 KTV 分轨增益
- 在音量调整与增益调整时，都会重新调用 `setMixerTrackGains`
- 避免出现设置项变化了，但 native mixer 仍沿用旧 gain 的情况

### 3. 跑马灯与歌词舞台

- 跑马灯不再因为“投影模式 / 纯黑背景”被顺手禁用
- 只有显式打开“低耗模式”时，才关闭颜色动画
- 新增颜色主题：
  - `gold`
  - `cyan`
  - `lime`
  - `sunset`
  - `arctic`
- 为歌词正文和扩展歌词增加更强的阴影描边，提升投影可读性

### 4. 排版与控件空间

- 主歌词 `lineHeight` 改为约等于字体实际高度
- 开启 `includeFontPadding: false`
- 减少每行上下 padding 与扩展歌词间距
- 压缩顶部栏和底部控制面板高度
- 调整横屏下按钮宽度，减少遮挡与重叠
