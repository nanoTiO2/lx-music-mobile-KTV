# 2026-04-20 13:05 循环重播延迟 / LRC入口 / 横屏标题修复说明

## 本轮问题

1. 单曲循环、随机播放在歌曲播完后出现数秒“假切歌”或进度时间异常。
2. 横屏播放页歌名与左上计时按钮发生重叠。
3. 本地歌曲在横屏与竖屏页面中均无法进入 LRC 歌词舞台页面。

## 本轮修改

### 1. 恢复本地歌曲进入 LRC 舞台

- 文件：`src/screens/PlayDetail/components/LyricStageBtn.tsx`
- 处理：
  - 移除“本地歌曲禁用歌词舞台入口”的误限制。
  - 现在本地歌曲与在线歌曲都可以打开歌词舞台页面。

### 2. 修复横屏标题与左上控件重叠

- 文件：`src/screens/PlayDetail/Horizontal/components/Header.tsx`
- 处理：
  - 给标题区域增加左侧留白与最小宽度保护。
  - 标题改为稳定左对齐，避免挤压后覆盖左上按钮区域。

### 3. 收紧播放结束事件链路

- 文件：`src/core/init/player/player.ts`
- 文件：`src/plugins/player/service.ts`
- 处理：
  - 单曲循环结束后不再走“先 seek 当前空白占位轨，再 play”的旧路径。
  - 改为直接复用现有 `playNext(true)` 切回当前曲目，减少结束后停留在占位轨上的时间。
  - 在 `PlaybackTrackChanged` / `PlaybackQueueEnded` 处增加结束事件收口与防重复触发保护。
  - 当切入占位轨时，立即暂停并回到 0 秒，减少用户看到“多出一首空白曲目”的窗口期。
